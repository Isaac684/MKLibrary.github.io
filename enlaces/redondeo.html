Redondeo de elementos Para hacer redondeo de elementos solo hay que llamar la clase rde-"tamaño" ya que hay varios tamaños de redondeo.
Elemento redondeado a tamaño S
Elemento redondeado a tamaño M
Elemento redondeado a tamaño L
Elemento redondeado a tamaño XL
Elemento redondeado a tamaño XXL
Codigo
<div class="rde-s p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado a tamaño S</div>
<div class="rde-m p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado a tamaño M</div>
<div class="rde-l p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado a tamaño L</div>
<div class="rde-xl p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado a tamaño XL</div>
<div class="rde-xxl p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado a tamaño XXL</div>
Redondeo de elementos por esquinas individuales Para redondear elementos por esquinas indiviuales hay que llamar a la clase rde-"esquina a redondear"-"tamaño" y asi redondear por cada esquina, ejemplo
Elemento redondeado de esquina superior izquierda
Elemento redondeado de esquina superior derecha
Elemento redondeado de esquina inferior izquierda
Elemento redondeado de esquina inferior derecha
Codigo
<div class="rde-tl-m p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado de esquina superior izquierda</div>
<div class="rde-tr-m p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado de esquina superior derecha</div>
<div class="rde-bl-m p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado de esquina inferior izquierda</div>
<div class="rde-br-m p-5 b-oscuro-1 sf-4 m-5">Elemento redondeado de esquina inferior derecha</div>
