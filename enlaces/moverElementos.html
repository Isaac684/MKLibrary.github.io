Posicionamiento de elementos

En MKLibrary es posible posicionar elementos en un total de 9 posiciones diferentes como son Arriba centrado, abajo centrado, centrado a la derecha y a la izquierda, arriba, abajo, a la derecha, a la izquierda etc

Para lograr posicionar un elemento, se debe colocar el elemento en un contenedor relative, el contenedor relative se genera de la siguiente manera, "pos-rel", ahora se debe declarar en donde se desea posicionar el elemento hijo en el archivo, la forma de llamar estas propiedades es la siguiente:

arriba: Tamaño <= 540px

abajo: Tamaño <= 720px

centrado: Tamaño <= 960px

izquierda: Tamaño <= 1200px

derecha: Tamaño <= 1400px

La propiedad centrado, tiene unas variaciones como lo son, centrado-izquierda, centrado-derecha, centrado-arriba, centrado-abajo. Las propiedades como abajo y derecha se pueden combinar para posicionar de otra forma el elemento, a continuacion una demostracion de esto:
